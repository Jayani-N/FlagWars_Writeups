# DotDash (50 pts)


## Description:

    Dot, Dash, Win! 
    
    -- --- .-. ... . ..--.- -- .- ... - . .-.
    
    Wrap the answer  in flagwars{}


## Solution:

As the chall name and desc says- dot,dash it's "Morse code", a classic cipher encoding which when decoded using sites like cyberchef.org gives the flag which shld be wrapped in the gvn format (flag is case-insensitive for this chall)
